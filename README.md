# 🧠 Professional Focus

I work on data-intensive systems where reliability, traceability, and data quality are critical.

My work focuses on:

• Data quality analysis and anomaly detection across large datasets  
• Development of automated data validation pipelines and testing frameworks  
• End-to-end testing of data processing systems and automation workflows  
• SQL-based analysis of large datasets using analytical databases and cloud platforms  
• Design of dashboards and analytics tools to monitor data quality and KPIs  
• Root-cause analysis of data inconsistencies through investigation of system logic and pipelines  
• Validation and monitoring of data migrations between enterprise systems  
• Definition of data structures, business rules, and governance principles for master data  
• Technical documentation of data architectures, validation workflows, and analytical processes
